About

Mulled is a repository of prebuilt Linux utilities, available as Docker containers from Quay.IO. In contrast to the usual approach with Dockerfiles, we strive for small (and fast) images.

Visit Source

How To Use

On this page, you can find all already built packages. They are all available from the mulled organization on the Quay organization page. The commands below are directly runnable and reference (via a SHA checksum) to the exact linked version.

Propose new package

To propose a new package, please fork our Github repository, and add the desired package to packages.tsv. We currently support packages from Homebrew, Alpine and Conda, but more will follow.
