Mulled Image Viewer

Mulled is a repository of prebuilt Linux utilities, available as Docker containers from Docker Hub. In contrast to the usual approach with Dockerfiles, we strive for small (and fast) images.

Propose new package

To propose a new package, please fork our Github repository, and add the desired package to packages.tsv. We currently support the following package managers, if you feel like one is missing, please get in touch:

  • Homebrew
