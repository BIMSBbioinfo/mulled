About

Mulled is a repository of prebuilt Linux utilities, available as Docker containers from Docker Hub. In contrast to the usual approach with Dockerfiles, we strive for small (and fast) images.

Visit Source

How To Use

On this page, you can find all already built packages. They are all available under the Docker Hub repository thriqon/mulled, the package name is the tag. However, for additional security, you can also use the digest next to the package. Using that you can be sure you get the image as it was produced on the build cluster.

Propose new package

To propose a new package, please fork our Github repository, and add the desired package to packages.tsv. We currently support packages from Homebrew, but more will follow.
